Procedure Text

Procedure Text

Procedure text is a text which aims to inform the readers about the steps to do something
The purpose of procedure text is to show how something is made, used, or done through a sequence of steps

Language Features of Procedure Text

To recognize a procedure text, we need to know its characteristics, just like when we meet someone for the first time. Here are some linguistic characteristics of procedure text:

  1. This text discusses facts about making or using something, so we need to use simple present tense.
  2. Because we need to explain some steps, we need to use imperative sentences. Examples of sentences in procedure text are: add 5 onions, get some yeast and pan. The most important thing is that the imperative sentence must be preceded by a verb. Examples are: mix, stir, boil, add, turn on, get, and so on.
  3. In procedure text, when we want to explain the steps of making/using, there are times when we need to connect one sentence with another. Therefore, we need connectives. For example: first, second, after that, then, finally, last, and so on.
  4. To give a description of the time or duration of the procedure text, we need to use adverbs. Examples are: in 5 minutes, for an hour, and so on. In addition, we can also use adverbs of manner. For example: slowly, fast, rapidly, powerfully, etc.

Generic Structures of Procedure Text

In creating procedural text, it is important to know what the generic structure is. The generic structure of a procedural text consists of aim/goal, ingredients/materials, and steps/methods.

Aim/Goal

The first part of writing a procedural text is the aim/goal, which means the goal. Usually, this goal is located in the title which shows the purpose of the text.

Example: How to Make Dalgona Coffee

Ingredients/Materials

In the second part, you are required to write ingredients/materials. Writing ingredients/materials is useful to explain what ingredients or tools are used to make something in the procedure text.

Example: The ingredients to make Dalgona coffee are coffee and sugar. We also need a spoon and warm water.

Steps/Methods

Ayaaay!!! We also arrived at the final part, which is writing steps/methods. This part contains what steps are taken to achieve the purpose of the procedure text.

Example: First, wash the tomatoes.
